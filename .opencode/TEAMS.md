## Team Workflows

### Feature Development
1. **Product**: Define requirements, prioritize
2. **UX/Design**: Research, wireframes, design, accessibility
3. **Security**: Review security implications
4. **Engineering**: Implement with tests
5. **QA**: Review, test, validate

### Bug Fix
1. **QA**: Report bug with details
2. **Engineering**: Investigate and fix
3. **Security**: Review for security implications
4. **QA**: Verify fix

### Security Review
1. **Security Backend**: Review authentication, authorization, SQL
2. **Security Frontend**: Review XSS, CSP, HTTPS
3. **Engineering**: Implement security fixes

### Accessibility Audit
1. **Accessibility**: Review WCAG compliance
2. **UX/Design**: Fix design issues
3. **Engineering**: Fix technical issues
