# Task Management Plan

Maintain tasks, track tasks execution, avoid forgetting tasks.

1. Task CRUD
2. Create Task Log
    - User can create task log (Day tasks or other name the user gives to it)
    - Move any task to day tasks or create new ones on the fly
    - Click on a task to make it active, other active tasks become background, active tasks
    - Click on an active, foreground task to make it inactive
    - Tracks time spent on the task
    - Click "None" to make no tasks active
    - Track full time in day tasks
    - Show visual charts of time spent on each task
    - allow classify tasks as preparatory, administrative, capability enhancement and end goal (feel free to suggest better naes)
