# Doce Lar - Home Management Tool

**Sua gestão financeira e cuidados em um só lugar**

A comprehensive home management application for Brazilian families.

## Core Modules

### Finance
- **Cards** - Multiple payment methods (credit/debit)
- **Purchases** - Expense tracking with installments
- **Payments** - Due date tracking
- **Products** - Price catalog

### Health
- **Patients** - Unified health records (People & Pets)
- **Treatments** - Treatment plans
- **Medications** - Prescription tracking

### Organization
- **Tasks** - Task management
- **Calendar** - Family calendar with unified events
- **Tags** - Organization system
- **Notes** - Personal notes

### Media
- **Galleries** - Photo organization
- **Articles** - Content management

### Learning
- **Library** - Family book collection
- **Reading** - Track reading progress and goals
- **Calibre** - Import from Calibre e-book library
- **Kindle** - Import from Kindle/Goodreads

## Principles

1. **Privacy First** - Your data stays yours
2. **No Lock-in** - Export anytime, anywhere
3. **Simplicity** - Easy for the whole family
4. **Transparency** - Clear costs and calculations

## Technology

- **Backend:** Ruby on Rails 8
- **Frontend:** Hotwire (Turbo + Stimulus)
- **CSS:** Modern CSS (no frameworks)
- **Database:** SQLite (portable)
- **Deployment:** Kamal (self-hosted)
